2.8a

Solve for a:

First quartile

\[F(x) = \int_0^{a} \lambda e^{-\lambda x} ~dx = .25\] \[-e^{-\lambda x} \rvert_{0}^{a} = .25\] \[1 - e^{-\lambda a} = .25\] \[e^{-\lambda a} = .75\] \[-\lambda a = ln(.75)\] \[a = \frac{ln(.75)}{\lambda}\]

Median

\[F(x) = \int_0^{a} \lambda e^{-\lambda x} ~dx = .5\] \[-e^{-\lambda x} \rvert_{0}^{a} = .5\] \[1 - e^{-\lambda a} = .5\] \[e^{-\lambda a} = .5\] \[-\lambda a = ln(.5)\] \[a = \frac{ln(.5)}{\lambda}\]

Third quartile

\[F(x) = \int_0^{a} \lambda e^{-\lambda x} dx = .75\] \[-e^{-\lambda x} \rvert_{0}^{a} = .75\] \[1 - e^{-\lambda a} = .75\] \[e^{-\lambda a} = .25\] \[-\lambda a = ln(.25)\] \[a = \frac{ln(.25)}{\lambda}\]

2.8b

Solve for b:

First quartile

\[F(x) = \int_{1}^{b} \frac{\alpha}{x^{\alpha + 1}} dx = .25\] \[\int_{1}^{b} x^{-\alpha - 1} dx = .25\] \[x^{-\alpha}\rvert_{0}^{b} = .25\] \[b^{\alpha} = .25\] \[b = \sqrt[a]{.25}\]

Median

\[F(x) = \int_{1}^{b} \frac{\alpha}{x^{\alpha + 1}} dx = .5\] \[\int_{1}^{b} x^{-\alpha - 1} dx = .5\] \[x^{-\alpha}\rvert_{0}^{b} = .5\] \[b^{\alpha} = .5\] \[b = \sqrt[a]{.5}\]

Third quartile

\[F(x) = \int_{1}^{b} \frac{\alpha}{x^{\alpha + 1}} dx = .75\] \[\int_{1}^{b} x^{-\alpha - 1} dx = .75\] \[x^{-\alpha}\rvert_{0}^{b} = .75\] \[b^{\alpha} = .75\] \[b = \sqrt[a]{.75}\]
